# Practica 03: Documentation Tests

## Objetivo

Escribir documentacion con ejemplos ejecutables.

## Instrucciones

1. Documenta las funciones con `///`
2. Agrega ejemplos en bloques de codigo
3. Usa las secciones estandar de rustdoc
4. Verifica que los doc tests pasen

## Secciones de Documentacion

- `# Arguments` - Parametros
- `# Returns` - Valor de retorno
- `# Example` - Ejemplo de uso
- `# Errors` - Errores posibles
- `# Panics` - Cuando produce panic

## Ejecutar

```bash
cargo test --doc
cargo doc --open
```
